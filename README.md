# The Siphon Interface

D&D 5E companion web app for the Siphon Wielder homebrew feature. Manages Echo Points, Focus, Siphon Capacitance, Echo Manifold phases, and the Select/Bestow/Activate card lifecycle.

## Quick Start

```bash
npm install
npm run dev
```

## Commands

| Command | Purpose |
|---------|---------|
| `npm run dev` | Start dev server |
| `npm run build` | Type-check + production build |
| `npm run lint` | ESLint |
| `npm run test` | Vitest (Phase 0+) |

## Tech Stack

React 19, TypeScript, Vite, Tailwind CSS v4, Zustand, React Router 7
